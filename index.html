Responsive Table

The standard table component used to display data in a tabular format.

Dependencies bootstrap jQuery.dotdotdot

Table Styling

Add all tables inside a div with .content-box class to show the white box container.

Table row as a link

Add class .clickable-row to display the pointer cursor when entire row is a link.

Drop Columns

Use bootstrap .hidden- and .visible- classes to hide and show columns according to viewport size.

Using ellipsis on table content

Use the scss mixin ellipsis to define line width and number of lines.

Possible values for width are: small, medium, large and full. There's no limit on number of lines.

Example:@include ellipsis(2,full); Ellipsis will happen when text grows more than 2 lines but will allow for full width according to container size.
